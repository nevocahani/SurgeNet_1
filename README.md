# SurgeNet v3.0 — מדריך הפעלה

## דרישות
- Python 3.13

## התקנה (פעם אחת בלבד)

```bash
cd surgenet
pip install -r requirements.txt
```

## הפעלה

```bash
python app.py
```

פתח דפדפן על: **http://localhost:5000**

## כניסה ראשונה

משתמש admin ראשוני:
- שם משתמש: `admin`
- סיסמה: `admin123`

## הוספת משתמשים

1. כנס כ-`admin`
2. לך ל"ניהול משתמשים"
3. הוסף ראש בית חולים עם שם משתמש בלבד
4. ראש בית חולים מוסיף ראשי מחלקות
5. ראשי מחלקות מוסיפים מנתחים
6. כל משתמש חדש מגדיר סיסמה בכניסה הראשונה

## קובץ הנתונים

הנתונים נשמרים ב: `surgenet.db` (SQLite)
